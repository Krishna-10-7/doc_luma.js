Introduction

LumaJS is designed for developers who want the power of modern frameworks without the bloat. Built from the ground up with performance in mind, it provides three core capabilities in a single, unified package.

React-like Reactivity
Hooks, components, and state management with a familiar API. Just 3KB for the core reactivity system.
GSAP-style Animations
30+ easing functions, timeline sequencing, and scroll triggers. Professional animations in just 7KB.
Three.js-like 3D
WebGL rendering with custom shaders, geometries, and camera controls. Full 3D engine in only 12KB.

Quick Start

Get up and running with LumaJS in under a minute.

import { h, mount, useState } from '@krishna.dev1/lumajs';

function Counter() {
  const [count, setCount] = useState(0);

  return h('div', {},
    h('h1', {}, `Count: ${count}`),
    h('button', { 
      onClick: () => setCount(count + 1) 
    }, 'Increment')
  );
}

mount(Counter, document.getElementById('app'));

Components

Build reusable UI components with props and composition.

import { h, mount } from '@krishna.dev1/lumajs';

// Simple component
function Greeting({ name }) {
  return h('div', { class: 'greeting' },
    h('h2', {}, `Hello, ${name}!`),
    h('p', {}, 'Welcome to LumaJS')
  );
}

// Composing components
function App() {
  return h('div', {},
    Greeting({ name: 'Alice' }),
    Greeting({ name: 'Bob' })
  );
}

mount(App, document.getElementById('root'));

3D Rendering

Create stunning 3D scenes with WebGL, all in just 12KB.

Basic 3D Scene

import { 
  Scene, PerspectiveCamera, WebGLRenderer, 
  Mesh, BoxGeometry, Material, OrbitControls 
} from '@krishna.dev1/lumajs/dist/luma-3d';

// Create scene and camera
const scene = new Scene();
const camera = new PerspectiveCamera(75, width / height, 0.1, 1000);
camera.position[2] = 5;

// Renderer
const renderer = new WebGLRenderer({ canvas, width, height });

// Cube
const geometry = new BoxGeometry(1, 1, 1);
const material = new Material({ 
  color: [0.5, 0.5, 1.0], 
  shaderType: 'phong' 
});
const cube = new Mesh(geometry, material);
scene.add(cube);

// Orbit controls
const controls = new OrbitControls(camera, renderer.canvas);

// Animation loop
function animate() {
  requestAnimationFrame(animate);
  cube.rotation[0] += 0.01;
  cube.rotation[1] += 0.01;
  controls.update();
  renderer.render(scene, camera);
}

Counter Application

import { h, mount, useState } from '@krishna.dev1/lumajs';

function App() {
  const [count, setCount] = useState(0);

  return h('div', { style: { padding: '20px' } },
    h('h1', {}, `Count: ${count}`),
    h('div', { style: { display: 'flex', gap: '10px' } },
      h('button', { onClick: () => setCount(count - 1) }, '-'),
      h('button', { onClick: () => setCount(0) }, 'Reset'),
      h('button', { onClick: () => setCount(count + 1) }, '+')
    )
  );
}

mount(App, document.getElementById('root'));

Timeline Animations

Chain multiple animations together with precise timing control.

import { timeline } from '@krishna.dev1/lumajs/dist/luma-animate';

const tl = timeline();

tl.to(box1, { x: 100 }, { duration: 500 })
  .to(box2, { y: 100 }, { duration: 500 })
  .to(box3, { rotation: 360 }, { duration: 1000 });

Scroll Triggers

Trigger animations based on scroll position.

import { scrollTrigger } from '@krishna.dev1/lumajs/dist/luma-animate';

scrollTrigger(element, {
  start: 'top center',
  end: 'bottom center',
  onEnter: () => {
    to(element, { opacity: 1, y: 0 });
  }
});
